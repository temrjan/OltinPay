## Как работаем

Роли: **Капитан** решает и ратифицирует, **Инженер** ведёт один трек за раз,
**Ревьюер** гейтит. Связь Инженер↔Ревьюер — **только через Капитана**.
Ревьюер в рабочее дерево не пишет (исключение — хэндоффы по прямому распоряжению
Капитана, как этот). Гейты: 1 (спека) и 2 (диф) для кода, один для ops.
Каждый PR: одна причина, доказательство приложено (для кода — красный тест,
для ops — чтение состояния «до/после»). **Мержим по зелёному CI.**
